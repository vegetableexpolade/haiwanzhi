# 志书编写海湾现场踏勘表（增强版）

## 本轮新增内容
- 新增管理员导出页，可导出“现场踏勘 / 现场调查 / 全部”Excel
- 新增 `exportSubmittedData` 云函数，导出已提交数据并生成 xlsx 文件
- 新增管理员权限字段 `isAdmin`
- `sendLoginCode` 已改成“正式短信 + 开发联调”双模式：
  - 配置好短信环境变量后，走腾讯云短信正式发送
  - 未配置时，自动退回开发联调 `devCode`

## 需要你在微信开发者工具中完成的事
1. 导入本项目
2. 配置自己的 AppID
3. 选择云环境 `cloud1-3gajf6tm046ec7c5`
4. 部署全部云函数（右键：上传并部署，且选择“云端安装依赖”）
5. （可自动创建，但建议提前初始化）新建数据库集合：
   - `users`
   - `loginCodes`
   - `segmentCounters`
   - `scoutSegments`
   - `surveyStations`
6. 初始化预置用户表 `users`
7. 对需要导出下载的管理员账号，在 `users` 集合补充 `isAdmin: true`
8. 在对象存储或云托管文件存储侧启用服务端加密策略

## 短信正式发送配置
`cloudfunctions/sendLoginCode` 支持腾讯云短信 SDK 正式发送。部署云函数时，请配置以下环境变量：

- `SMS_ENABLED=true`
- `SMS_SDKAPPID=你的短信SdkAppId`
- `SMS_SIGN=已审核通过的短信签名`
- `SMS_TEMPLATE_ID=已审核通过的验证码模板ID`
- `SMS_REGION=ap-guangzhou`（可选）

说明：云函数中调用腾讯云 SDK 时，通常还需要运行角色已具备短信权限，或提供可用的临时密钥环境变量。

## 关于 Excel 导出
管理员可在“现场踏勘和现场调查”页面中看到“管理员导出”入口。导出后会：
1. 云函数生成 xlsx
2. 上传到云端 `exports/` 目录
3. 返回临时下载链接
4. 小程序自动下载并尝试打开文档

## 关于照片 GPS
微信小程序前端不能稳定保证每张图片都自带可直接用于 Google Earth 的 EXIF GPS，因此本项目采取的是：
- 拍照瞬间先调用定位
- 将经纬度、拍照时间与照片元数据一起保存

若你需要把经纬度真正写回图片 EXIF，建议在后台增加二次处理服务。


## 第三轮新增
- 新增管理员“注册审批”页面，可查看待审批用户、通过或驳回注册申请
- 新增 `listApprovalUsers`、`reviewUser` 云函数

## 审批流程建议
1. 用户先提交注册申请
2. 管理员进入“注册审批”页面核对预置登记表
3. 审批通过后，用户先用短信验证码登录
4. 第一次登录时强制修改密码
